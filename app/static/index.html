Welcome to Our API

This page provides basic details about our REST API and WebSocket functionalities.

REST API

  • POST /chats: Create a new chat session. Returns a unique chat_id.
  • PUT /chats/{chat_id}: Send a message to a chat session and get a response.

For more details, visit our Swagger Documentation.

WebSocket

For real-time chat capabilities, we offer WebSocket communication.

  • Connect via: ws://your_domain/chats/{chat_id}

Remember to provide the API key for authentication purposes.

How to Use

  1. Create a chat session using the REST API's POST endpoint.
  2. Use the returned chat_id to interact using either REST or WebSocket methods.
